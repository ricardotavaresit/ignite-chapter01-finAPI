## Quick start

- Clone the repo: `git clone https://github.com/ricardotavaresit/ignite-chapter01-finAPI.git`

- Install dependencies: `npm install` or `yarn`

- Start the server: `npm run start` or `yarn start`

- Data are on: `localhost:3333`
